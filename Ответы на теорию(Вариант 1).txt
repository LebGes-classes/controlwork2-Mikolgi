Вариант 1.
1.Что такое ооп? ООП - это подход, при котором программа состоит из набора объектов, которые взаимодействуют друг с другом. Каждый имеет свои свойства и поведение.
2. Основные принципы ООП:
1)Инкапсуляция
2)Полиморфизм - это когда один и тот же метод может работать по-разному в зависимости от объекта.  
3)Наследование - это когда подкласс берет свойства класса и добавляет свои.
4)Абстракция - это когда выделяешь общие характеристики объекта, свойства, методы, но не реализуешь их. Описываешь только ключевое в общих чертах
3. Инкапсуляция - это когда вся информация для объекта, хранится внутри самого объекта. Если нужно что-то изменить, то нужно использовать методы внутри объекта. Так же используются сеттеры и геттеры, это методы для получения информации, ее изменения извне, не получая пряимого доступа к полям.
4. Класс - это шаблон, на базе которого можно построить какой-нибудь объект. Например, есть класс машина, на ее основе можно создать объекты - конкретные машины.
5. Отличие между классом и объектом в том, что класс шаблон, которые описывает поведение и характеристики, которые будет использовать объект. Объект - это экземпляр класса. Например, есть класс машина, у нее есть определенное поведение: гудить, ездить. А объект - это конкретная машина, которая гудит определенным образом и едет определенным образом.
